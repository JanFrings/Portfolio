    Thank you for visiting!

    Feel free to check out my projects.
    Each one had the purpose to push my skills to the edge of what I was able to do at that point.
    You'll find a short description of each project along with the technical knowledge I had to acquire.

    Website Project

    #Build a static website


    A closer look at web development revealed that a basic understanding of different coding languages was indispensable.
    • HTML, CSS, JavaScript

    To make these languages in different files work together as one website, a framework technology and a package manager was needed.
    • Django, Anaconda

    The small progress I made each day had to be saved and documented somewhere, that's when version control came in handy.
    • GitHub

    Finally, a deployment service was necessary to make the website accessible worldwide.
    • DeploymentXX

    Blog Application

    #Add functionality


    The step from a static website, that only displays information, to a dynamic platform that allows interaction had to be made.
    This requires a back-end understanding of how to save data to a database and query it when needed.
    • Python

    NFL Project

    #Collect, filter, manipulate data


    Processing data is one of the most important skills a software engineer has to have.

    First I needed to collect meaningful data which I did with the help of a scraper tool.
    • Beautiful Soup

    The raw scraped data had to be filtered. I used software that supports interactive data science and scientific computing.
    • Jupyter Notebook

    The concluding step was to manipulate the filtered data in such a way that insights are easily displayed and understood. I used open source data science packages for that purpose.
    • NumPy, Pandas, Matplotlib, Seaborn, Stats Models, Scikit learn
